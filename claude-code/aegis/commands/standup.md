Prefer the plugin skill `/aegis:aegis-operator` when available.

Call `aegis_mcp_status` if readiness is unknown, then use `aegis_team_health`,
`aegis_nudges`, and `aegis_report` with `view=standup`. Give a concise briefing:
in flight, blocked/at risk, owner decisions, and the one thing you recommend
moving next. Stay strictly grounded in tool output.
